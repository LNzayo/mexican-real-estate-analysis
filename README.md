# 🏠 Mexican Real Estate Market Analysis
## Hands-on Data Science Project - WorldQuant University

Data science analysis of 35,000+ Mexican properties. Explores how location, size, and property type influence pricing. Key finding: Location explains 35% of price variation vs. 23% for size. Built with Python, Pandas, Matplotlib, Seaborn.

### 📌 Project Overview
This project is part of the **Applied Data Science Lab** at **WorldQuant University**. The goal was to analyze the Mexican real estate market using data science techniques to understand what drives property prices.

### 🔍 Research Question
> **Is the size of a property (`area_m2`) a stronger predictor of price than its location (`state`)?**

### 📊 Dataset
      - **Source**: Mexico Real Estate Combined Clean Dataset
- **Records**: ~35,000 properties
- **Key Features**: `price_usd`, `area_m2`, `state`, `property_type`

### 🛠️ Tools & Technologies
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook
- Git & GitHub

### 💡 Key Findings
1. **Location explains 35.2%** of price variation vs. **23.2%** for size
2. **Diminishing returns**: Larger properties cost less per m²
3. **State variation**: Prices range from $890/m² to $1,850/m²
4. **Property type**: Houses are most expensive ($1,850/m²), Studios are cheapest ($1,420/m²)

### 🎯 Conclusion
> **Location matters more than size in the Mexican real estate market.**
